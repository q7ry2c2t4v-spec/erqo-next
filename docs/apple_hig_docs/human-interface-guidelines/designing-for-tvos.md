# tvOS向けのデザイン

tvOSでは、メディアやゲームのほか、フィットネス、教育、ホームユーティリティのアプリで提供される活気あふれるコンテンツ、イマーシブ感のある視聴体験、スムーズな操作を楽しむことができます。

![図案化されたテレビの画面。グリッドの一番上に表示されています。画像では長方形と円形のグリッド線が重なっており、6色のオリジナルAppleロゴをイメージさせる緑色になっています。](https://docs-assets.developer.apple.com/published/62fca31d9b9015ac9e1cc9cc379c6919/platforms-tvOS-intro%402x.png)

tvOS向けのアプリやゲームをデザインするときは、まず以下に示すデバイスの基本的な特徴やtvOSならではの操作パターンを理解することから始めてください。この特徴やパターンを参考にしてデザインを決定することで、tvOSユーザに喜ばれるアプリやゲームを提供できます。

**ディスプレイ:** 一般的にテレビには超大型サイズの高解像度ディスプレイが搭載されています。

**人間工学:** 据え置きのテレビから数メートル（通常は2.5メートル以上）離れるのが一般的ですが、部屋の中を動き回りながらコンテンツを操作することもあります。

**入力:**[リモコン](https://developer.apple.com/jp/design/human-interface-guidelines/remotes)、[ゲームコントローラ](https://developer.apple.com/jp/design/human-interface-guidelines/game-controls)、[Siri](https://developer.apple.com/jp/design/human-interface-guidelines/siri)や、ほかのデバイスで動作しているアプリを使用してApple TVを操作できます。

**アプリの操作:** 1つの体験（数時間継続するものが多い）に深く集中したり、ピクチャインピクチャ表示にして同時に別のアプリやビデオを見たりすることもできます。

**システム機能:** アプリle TVのユーザは、アプリやゲームに以下のシステム体験が組み込まれていることを期待します。

  * [TVアプリへの対応](/jp/design/human-interface-guidelines/playing-video#Integrating-with-the-TV-app)

  * [SharePlay](/jp/design/human-interface-guidelines/shareplay)

  * [トップシェルフ](/jp/design/human-interface-guidelines/top-shelf)

  * [TVプロバイダのアカウント](/jp/design/human-interface-guidelines/managing-accounts#TV-provider-accounts)

## [ベストプラクティス](/jp/design/human-interface-guidelines/designing-for-tvos#Best-practices)

tvOSでの優れた体験は、ユーザにとって最も価値のあるプラットフォームとデバイスの機能を統合することで実現されます。tvOSらしい操作環境にするために、以下の方法に優先順位を付けてそれぞれの特徴や機能を取り入れてください。

  * Siri Remoteでの馴染みのあるスムーズなジェスチャを使用して、パワフルで楽しい操作に対応します。

  * tvOSのフォーカスシステムに対応させます。これにより、ユーザが画面上の項目を移動する際に、少しだけハイライトされて拡大表示されるので、今どこにいて何をすべきかが常に分かるようになります。

  * 画面の隅々まで活用した美しいアートワーク、スムーズで精巧なアニメーション、魅力的なオーディオを提供します。部屋のどこから見てもはっきりと文字を読みやすくして、人の心を魅了する映画のような豊かな体験でユーザを包み込みます。

  * サインインを簡単にして回数を減らしたり、共有サインインに対応したり、現在の視聴者を変更したときに自動的にプロファイルを切り替えたりすることで、マルチユーザ対応を強化します。
