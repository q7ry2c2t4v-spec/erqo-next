# イメージウェル

イメージウェルは、編集可能なバージョンの画像ビューです。

![図案化されたイメージウェル。画像は6色のオリジナルAppleロゴをイメージさせる赤色になっています。](https://docs-assets.developer.apple.com/published/a7e51c30befc44045cdb1777d000a53f/components-image-well-intro%402x.png)

イメージウェルを選択すると、ユーザがイメージウェルの画像をコピー&ペーストしたり、削除したりできます。また、イメージウェルをあらかじめ選択せずに、中に新しい画像をドラッグすることもできます。

## [ベストプラクティス](/jp/design/human-interface-guidelines/image-wells#Best-practices)

**必要に応じてデフォルト画像に戻す。** イメージウェルに画像が必要な場合は、ユーザがイメージウェルのコンテンツを消去したときに再度デフォルト画像を表示します。

**イメージウェルをコピー &ペーストに対応させる場合は、標準のコピー&ペーストのメニュー項目を使用できるようにする。**一般に、ユーザはイメージウェルを操作するために、これらのメニュー項目を選択する（または標準のキーボードショートカットを使用する）ことを予期します。ガイダンスは[編集メニュー](/jp/design/human-interface-guidelines/the-menu-bar#Edit-menu)を参照してください。

関連するガイダンスは、[画像ビュー](/jp/design/human-interface-guidelines/image-views)を参照してください。

## [プラットフォームの考慮事項](/jp/design/human-interface-guidelines/image-wells#Platform-considerations)

 _iOS、iPadOS、tvOS、visionOS、watchOSには対応していません。_
