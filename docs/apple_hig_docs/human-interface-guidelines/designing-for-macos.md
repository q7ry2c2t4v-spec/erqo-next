# macOS向けのデザイン

Macは、綿密な制作作業を行ったり、メディアやコンテンツを視聴したり、ゲームをプレイしたりするのに十分対応できるパワー、画面の広さ、柔軟性を備えています。多くの場合、複数のアプリが同時に使用されます。

![図案化されたMacのフレーム。グリッドの一番上に表示されています。画像では長方形と円形のグリッド線が重なっており、6色のオリジナルAppleロゴをイメージさせる緑色になっています。](https://docs-assets.developer.apple.com/published/aea0ab6441ad8b24739a912825b272a2/platforms-macOS-intro%402x.png)

macOS向けのアプリやゲームをデザインするときは、まずデバイスの基本的な特徴やmacOSならではの操作パターンを理解することから始めてください。この特徴やパターンを参考にしてデザインを決定することで、Macユーザに喜ばれるアプリやゲームを提供できます。

**ディスプレイ:** 通常、Macには大型サイズの高解像度ディスプレイが搭載されています。iPadなどの追加ディスプレイを接続してワークスペースを拡張することもできます。

**人間工学:** 通常は場所を移動せずに机やテーブルなどに置いてMacを使用します。一般的な使い方では、視聴距離は30センチから90センチほどです。

**入力:** 物理[キーボード](/jp/design/human-interface-guidelines/keyboards)、[ポインティングデバイス](/jp/design/human-interface-guidelines/pointing-devices)、[ゲームコントロール](/jp/design/human-interface-guidelines/game-controls)、[Siri](/jp/design/human-interface-guidelines/siri)などの入力モードを組み合わせてデータの入力やインターフェイスの操作を行うことが想定されます。

**アプリの操作:** 短いタスクが数分で行われることも、集中して数時間のタスクが行われることもあります。多くの場合、同時に複数のアプリが開かれます。アプリを切り替えるときにはアクティブと非アクティブの状態がスムーズに遷移することが期待されます。

**システム機能:** macOSには、馴染みのある一貫した方法でシステムやアプリを操作できる複数の機能があります。

  * [メニューバー](/jp/design/human-interface-guidelines/the-menu-bar)

  * [ファイルの管理](/jp/design/human-interface-guidelines/file-management)

  * [フルスクリーンモード](/jp/design/human-interface-guidelines/going-full-screen)

  * [Dockメニュー](/jp/design/human-interface-guidelines/dock-menus)

## [ベストプラクティス](/jp/design/human-interface-guidelines/designing-for-macos#Best-practices)

Macでの優れた体験は、ユーザにとって最も価値のあるプラットフォームとデバイスの機能を統合することで実現されます。macOSらしいデザインにするために、以下の方法に優先順位を付けてそれぞれの特徴や機能を取り入れてください。

  * 大型ディスプレイを活用し、入れ子のレベルを少なくして表示するコンテンツを増やしつつ、モーダル表示を少なくします。また、負担なく目的のコンテンツを見ることができるように、適度な情報密度にします。

  * 作業スタイルやデバイスの設定に合うように、ウインドウのサイズ変更、非表示、表示、移動ができるようにします。また、気が散ることがないように、フルスクリーンモードに対応します。

  * メニューバーを使用して、アプリで必要なことを実行するすべてのコマンドに簡単にアクセスできるようにします。

  * ピクセル単位の選択や編集ができるように、高精度の入力モードを利用できるようにします。

  * 素早くアクションを実行できるように、またキーボードのみの作業スタイルを使用できるように、キーボードショートカットに対応します。

  * ツールバーをカスタマイズできるようにする、よく使うビューを表示するようにウインドウを設定できるようにする、インターフェイスに表示するカラーやフォントを選択できるようにするなど、パーソナライズに対応させます。
