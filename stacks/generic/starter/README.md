# プタ子プロジェクト

erqo-next OS (プタ子版) がインストールされました。

## 次のステップ

スタック依存のプロジェクト初期化は手動で行ってください:

- **Flutter** — `flutter create .`
- **Rust** — `cargo init`
- **Go** — `go mod init <module-path>`
- **その他** — 各スタックのドキュメントに従う

## プタ子で使えるスキル

- `/dsgn` — 設計
- `/tp` — タスク計画
- `/codi` — 実装パイプライン (通常タスク)
- `/rsrc` — ウェブリサーチ
- `/wrap` — セッションログ + git コミット
- `/ret` — 前回セッションの復元
- `/aud` — 作業監査
- `/ruru` — プロジェクトルール管理
- `/fb` — erqo-next へのフィードバック送信

※ `/layo` (レイアウトデザイン策定) は Next.js 特化装備のためプタ子には配布されません。

後続の実装作業は `/dsgn` から通常通り進められます。
