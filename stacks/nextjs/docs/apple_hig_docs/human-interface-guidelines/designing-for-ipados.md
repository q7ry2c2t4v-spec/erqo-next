# iPadOS向けのデザイン

iPadは、メディアを視聴したり、ゲームをプレイしたり、細かい制作作業を行ったり、制作物に命を吹き込んだりする場面でのパワー、持ち運びのしやすさ、柔軟性に優れています。

![図案化されたiPadのフレーム。グリッドの一番上に表示されています。画像では長方形と円形のグリッド線が重なっており、6色のオリジナルAppleロゴをイメージさせる緑色になっています。](https://docs-assets.developer.apple.com/published/b3dfbad024eebc26e21b52cef8b2d9e9/platforms-iPadOS-intro%402x.png)

iPad向けのアプリやゲームをデザインするときは、まず以下に示すデバイスの基本的な特徴やiPadOSならではの操作パターンを理解することから始めてください。この特徴やパターンを参考にしてデザインを決定することで、iPadユーザに喜ばれるアプリやゲームを提供できます。

**ディスプレイ:** iPadには大型サイズの高解像度ディスプレイが搭載されています。

**人間工学:** 通常、iPadは手で持って操作しますが、何かの上に置いたり、スタンドを使用したりすることもあります。一般的にデバイスを操作するときの視聴距離は90センチ以内ですが、デバイスの設置方法によって視聴距離が変わる可能性があります。

**入力:** iPadは、Multi-Touch対応の[ジェスチャ](https://developer.apple.com/jp/design/human-interface-guidelines/gestures)と[仮想キーボード](https://developer.apple.com/jp/design/human-interface-guidelines/virtual-keyboards)、接続した[キーボード](https://developer.apple.com/jp/design/human-interface-guidelines/keyboards)や[ポインティングデバイス](https://developer.apple.com/jp/design/human-interface-guidelines/pointing-devices)、[Apple Pencil](https://developer.apple.com/jp/design/human-interface-guidelines/apple-pencil-and-scribble)、[音声](https://developer.apple.com/jp/design/human-interface-guidelines/siri)で操作できます。複数の入力モードを組み合わせて使うのが一般的です。

**アプリの操作:** iPadではいくつかの短いアクションがときどき行われることもあれば、ゲーム、メディア、コンテンツ作成、制作作業に何時間も集中することもあります。多くの場合は、同時に複数のアプリを開き、複数のアプリを画面に表示させて、ドラッグ&ドロップのようなアプリ間機能がよく利用されます。

**システム機能:** iPadOSには、馴染みのある一貫した方法でシステムやアプリを操作できる複数の機能があります。

  * [マルチタスク](/jp/design/human-interface-guidelines/multitasking)

  * [ウィジェット](/jp/design/human-interface-guidelines/widgets)

  * [ドラッグ＆ドロップ](/jp/design/human-interface-guidelines/drag-and-drop)

## [ベストプラクティス](/jp/design/human-interface-guidelines/designing-for-ipados#Best-practices)

iPadでの優れた体験は、ユーザにとって最も価値のあるプラットフォームとデバイスの機能を統合することで実現されます。iPadOSらしい操作環境にするために、以下の方法に優先順位を付けてそれぞれの特徴や機能を取り入れてください。

  * 大型ディスプレイのメリットを活かして、ユーザが関心を持つコンテンツを上に置きます。モーダルインターフェイスやフルスクリーンの画面遷移は最低限にとどめ、画面上のコントロールを手が届きやすく邪魔にならない場所に配置します。

  * 視聴距離と入力モードを参考にして、画面に表示するコンテンツの大きさや密度を決めます。

  * Multi-Touch対応のジェスチャ、物理キーボードや物理トラックパッド、Apple Pencilを利用できるようにして、複数の入力モードを組み合わせた独自の操作に対応することを検討します。

  * デバイスの回転、マルチタスクモード、ダークモード、ダイナミックタイプなどの表示方法の変化や、macOSへの転送にシームレスに適応させて、ユーザが自分に最適な設定を選べるようにします。
