# NFC

近距離無線通信（NFC）は、数cmの距離にあるデバイス間でのワイヤレスな情報交換を可能にする技術です。

![右に向かって徐々に大きくなる曲線のスケッチ。近距離無線通信を示しています。画像では長方形と円形のグリッド線が重なっており、6色のオリジナルAppleロゴをイメージさせる青色になっています。](https://docs-assets.developer.apple.com/published/ba30b4185619a952d57ef40aad30f316/technologies-nfc-intro%402x.png)

対応デバイス上のiOSアプリでは、NFCスキャンを使って、現実世界の対象物に取り付けられた電子タグからデータを読み取ることができます。例えば、玩具をスキャンしてビデオゲームとリンクする、店頭のPOPをスキャンしてクーポンを受け取る、小売店の従業員として商品をスキャンして在庫をトラッキングする、といったことが可能です。

## [アプリ内でのタグ読み取り](/jp/design/human-interface-guidelines/nfc#In-app-tag-reading)

アプリでは、アプリの起動中に1つまたは複数の対象物をスキャンすることができ、ユーザにスキャンしてもらうタイミングでスキャンシートを表示することができます。

![iPhoneのスキャンシートの図。「スキャンの準備ができました」および「iPhoneをNFCタグにかざしてください」というテキストと「キャンセル」ボタンがあります。](https://docs-assets.developer.apple.com/published/ba1a90e2ffb72abf64dc04f09c6c0a63/nfc-ready-to-scan%402x.png)

**スキャン対象物に触れるように促さない。** タグをスキャンするには、iOSデバイスをタグに近づけるだけで十分で、タグを実際にタッチする必要はありません。スキャンを促すときには「 _タップする_ 」や「 _タッチする_ 」のような言葉は使わず、「 _スキャンする_ 」や「 _かざす_ 」などの言葉を使ってください。

**ユーザに馴染みのある表現を使う。** 近距離無線通信（NFC）という言葉を聞いたことがない人もいるかもしれません。ユーザにとって分かりやすくするために、デベロッパ向けの専門的な用語（ _NFC_ 、 _Core NFC_ 、 _近距離無線通信_ 、 _タグ_ など）は避けてください。代わりに、大半のユーザが理解できる、分かりやすい日常用語を使いましょう。

OK| NG  
---|---  
[_対象物の名前_]をスキャンしてください。| NFCタグをスキャンしてください。  
詳細を確認するにはiPhoneを[_対象物の名前_]にかざしてください。| NFCスキャンを使用するには、[_対象物の名前_]をiPhoneでタップしてください。  
  
**スキャンシートの説明文を簡潔にする。** 完全な文の形にし、文末記号を付けます（英語の場合はセンテンススタイルで大文字化します）。スキャン対象物を明確にし、2回目以降のスキャンでは適宜文言を変えます。文を短くし、テキスト切れが生じないようにしてください。

1回目のスキャン| 2回目以降のスキャン  
---|---  
詳細を確認するにはiPhoneを[_対象物の名前_]にかざしてください。| iPhoneを別の[_対象物の名前_]にかざしてください。  
  
## [バックグラウンドのタグ読み取り](/jp/design/human-interface-guidelines/nfc#Background-tag-reading)

バックグラウンドのタグ読み取りは、アプリを開いてスキャンを開始するという操作をせずに、いつでも素早くタグをスキャンすることができる機能です。バックグラウンドのタグ読み取りに対応しているデバイスでは、画面の点灯中にシステムが付近の互換タグを自動的に探します。タグが検出され、タグとアプリの照合が完了すると、タグデータをアプリに渡して処理するためにユーザにタップを求める通知が表示されます。ただし、NFCスキャンシートの表示中や、ウォレットまたはApple Payの使用中、カメラの使用中、機内モード中、およびデバイスが再起動後でロック中の場合は、バックグラウンドのタグ読み取りが無効になるので注意してください。

![iPhoneのホーム画面の上に表示された通知バナーの図。ここから、付近で検出されたNFCタグデータを処理する特定のアプリを開けるようになっています。](https://docs-assets.developer.apple.com/published/3db4e1e04f516a59c3e99051d894f9a7/nfc-background%402x.png)

**バックグラウンドとアプリ内の両方のタグ読み取りに対応する。** バックグラウンドのタグ読み取りを実装した場合でも、それに対応していないデバイスを使っているユーザのために、アプリ内でタグをスキャンできる手段を提供する必要があります。

## [プラットフォームの考慮事項](/jp/design/human-interface-guidelines/nfc#Platform-considerations)

 _iOS、iPadOSに追加の考慮事項はありません。macOS、tvOS、visionOS、watchOSには対応していません。_
