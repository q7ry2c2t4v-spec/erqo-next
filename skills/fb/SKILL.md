# /fb — フィードバック送信スキル

## 概要

erqo-nxt への改善提案・エラー報告・その他のフィードバックを送信する。

## いつ使うか

- erqo-nxt の動作に問題があったとき
- 改善のアイデアがあるとき
- その他のフィードバックを送りたいとき

## 実行手順

### 1. ユーザーとの対話

フィードバックの内容を確認する:
- **種類**: suggestion (改善提案) / error (エラー報告) / other (その他)
- **概要**: 1行で内容を要約
- **詳細**: (任意) 詳しい説明

### 2. フィードバック送信

```bash
python "{nxt}/skills/fb/handler.py" <kind> <summary> [detail]
```

例:
```bash
python "{nxt}/skills/fb/handler.py" suggestion "インデックス検索の精度改善" "タグの部分一致が効かない場面がある"
```

### 3. 結果確認

保存先のパスが表示される。erqo-nxt 本体のセッション開始時に通知される。

## 重要なルール

1. **ユーザーの承認** — フィードバック内容をユーザーに確認してから送信する
2. **種類の選択** — 内容に合った kind を選ぶ (suggestion / error / other)
